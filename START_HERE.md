# راهنمای استفاده با Claude Code

این پوشه شاملِ فرانت‌اندِ کاملِ فروشگاهِ «هور» + سندِ ساختِ بک‌اند است.

## قدم ۱ — نصب Claude Code
روی کامپیوترِ خودت (در ترمینال):
```bash
npm install -g @anthropic-ai/claude-code
```
سپس وارد شو:
```bash
claude
```
(در اولین اجرا، حسابِ Anthropic می‌خواهد.)

## قدم ۲ — باز کردن پروژه
ترمینال را در همین پوشه (پوشه‌ی پروژه‌ی هور) باز کن و `claude` را اجرا کن.

## قدم ۳ — دادن دستور
این جمله را به Claude Code بگو:

> «فایلِ `BACKEND_HANDOFF.md` را بخوان. می‌خواهم برای این فرانت‌اند یک بک‌اندِ واقعی بسازی.
> از Phase 1 شروع کن، هر مرحله را کامل کن و قبل از رفتن به مرحله‌ی بعد، تست کن و از من تأیید بگیر.»

## قدم ۴ — همراهی
Claude Code کد را می‌نویسد، ولی تو باید:
- وقتی چیزی می‌پرسد (مثل اطلاعاتِ دیتابیس یا کلیدِ درگاه) جواب بدهی
- خروجی هر مرحله را در مرورگر تست کنی
- خطاها را به او گزارش بدهی تا رفع کند

## نکته
- برای **درگاه پرداخت** و **پیامک** به حساب‌های واقعی نیاز است؛ در فاز توسعه از حالتِ سندباکس/تست استفاده کن.
- کارهای حقوقی (اینماد، ثبتِ کسب‌وکار) را موازی پیش ببر — به کد ربطی ندارند ولی برای آنلاین‌شدن لازم‌اند.
