یک نمونه‌ی اولیه‌ی تعاملی (prototype) با React — کاملاً فارسی و RTL — شاملِ فروشگاه، صفحه‌ی محصول، سبد خرید، شبیه‌سازیِ درگاه پرداخت، علاقه‌مندی‌ها، پنل مشتری و پنل مدیریت.

## ✨ امکانات

- 🛍 **فروشگاه** با دسته‌بندیِ تصویری (مردانه / زنانه / مشترک)
- 👓 **صفحه‌ی محصول** با گالریِ عکس، مشخصات و محصولات مرتبط
- 🛒 **سبد خرید و پرداختِ چندمرحله‌ای** با شبیه‌سازیِ درگاه بانکی
- ❤️ **علاقه‌مندی‌ها** (Wishlist)
- 👤 **پنل مشتری** (مشخصات، نشانی، سابقه و پیگیریِ سفارش)
- ⚙️ **پنل مدیریت** (افزودن/ویرایش/حذفِ محصول، مدیریت سفارش‌ها و مشتریان)
- 🎬 هیروی گالری + ویدئوی تیزر
- 🎨 پنل **تنظیماتِ ظاهر** (رنگ، پس‌زمینه، فرمِ گوشه‌ها)
- 📱 کاملاً **واکنش‌گرا** (موبایل تا دسکتاپ)

## اجرای محلی (Local)

این پروژه **استاتیک** است و نیازی به build ندارد. فقط باید با یک وب‌سرورِ ساده اجرا شود (به‌خاطرِ بارگذاریِ فایل‌های `.jsx`).

ساده‌ترین راه‌ها:

```bash
# روش ۱: با پایتون
python3 -m http.server 8000
# سپس باز کن: http://localhost:8000

# روش ۲: با Node
npx serve
```

یا در VS Code از افزونه‌ی **Live Server** استفاده کن.

> توجه: باز کردنِ مستقیمِ `index.html` با دابل‌کلیک (پروتکلِ file://) ممکن است به‌خاطرِ محدودیتِ CORS کار نکند؛ حتماً از یک سرورِ محلی استفاده کن.

## تکنولوژی‌ها

- **React 18** + **Babel** (ترجمه‌ی JSX در مرورگر)
- **CSS خالص** (CSS Variables, Grid, Flexbox) — کاملاً RTL
- فونت‌های **Dana** و **Vazirmatn**
- ذخیره‌سازی در **localStorage**
- بدون فریم‌ورک یا build-tool

## ساختار فایل‌ها

| فایل | توضیح |
|------|-------|
| `index.html` | فایل اصلی و بارگذاریِ اسکریپت‌ها |
| `styles.css` | سیستم طراحی (رنگ، فونت، کامپوننت‌ها) |
| `data.jsx` | داده‌ها (محصولات، دسته‌ها) |
| `ui.jsx` | آیکون‌ها و اجزای پایه |
| `components.jsx` | هدر، فوتر، کارت محصول، سبد |
| `pages1.jsx` | خانه، فروشگاه، صفحه‌ی محصول |
| `pages2.jsx` | سبد، پرداخت، علاقه‌مندی‌ها |
| `pages3.jsx` | پنل مشتری، پشتیبانی، درباره |
| `admin.jsx` | پنل مدیریت (CRUD محصولات) |
| `app.jsx` | مدیریت state و مسیریابی |
| `tweaks-panel.jsx` | اجزای پایه‌ی پنل تنظیمات |
| `theme-tweaks.jsx` | پنل تنظیماتِ ظاهر |
| `assets/` | عکس‌ها، لوگو و ویدئوی تیزر |
